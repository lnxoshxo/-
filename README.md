# 上实服务物业集团展会介绍系统

这是一个适用于触摸屏设备的展会介绍系统示例项目，面向上实服务物业集团在物业展会中的品牌展示与 AI 问答场景。

当前项目同时保留 3 个前台版本入口：

- 第一版：展会交互展示版，入口 `/`
- 第二版：极简高级商务首页版，入口 `/v2`
- 第二版 2.1：颗粒材质增强版，入口 `/v2-1`
- 第二版 2.2：背景重构版，入口 `/v2-2`
- 第二版 2.3：纯动态颗粒版，入口 `/v2-3`
- 第二版 2.4：细腻纯动态颗粒版，入口 `/v2-4`

## 功能概览

- 首页品牌展示：包含企业标识、科技感主视觉、主题标语与模块导航
- AI 知识库问答：用户输入问题后，后端在 JSON 知识库中执行相似度检索并返回答案
- 语音交互：支持浏览器语音输入与语音播报开关
- 展示模块：内置 `集团概况`、`服务项目`、`科技创新`、`未来展望` 等展区内容
- 历史记录：保存最近 5 条提问，并支持点击重新提问
- 推荐问题：支持预设高频问题，便于展会现场快速演示
- 待机轮播：支持大屏无人操作时自动进入吸引模式
- 自动讲解：支持按预设脚本自动滚动讲解与语音播报
- 管理后台：提供 `/admin` 页面直接编辑和导入 JSON 知识库
- 品牌接入预留：支持文字 Logo 与图片 Logo 切换、本地图片/视频占位替换
- LLM 接入预留：已预留基于知识库的扩展模式说明
- 参展欢迎页：支持欢迎屏保、时间日期显示与双入口模式
- 默认数据恢复：后台支持一键恢复默认演示数据
- 响应式布局：兼容大屏触摸设备与常规平板、桌面浏览器
- 多版本并行：支持第一版、第二版、第二版 2.1 同时在线预览

## 知识库结构说明

知识库文件位于 `data/knowledge_base.json`，主要分为两个部分：

- `sections`：用于首页各展示板块
- `entries`：用于 AI 问答搜索匹配
- `branding`：用于 Logo 与品牌展示配置
- `quick_questions`：用于推荐提问按钮
- `idle_slides`：用于待机轮播展示内容
- `guided_tour`：用于自动讲解流程与播报脚本

`entries` 单项结构示例：

```json
{
  "title": "科技创新",
  "summary": "集团正推进 AI、物联网和数字化平台在物业运营中的落地。",
  "keywords": ["科技", "创新", "AI", "物联网"],
  "highlights": [
    "部署 AI 视频巡检辅助秩序与安全管理",
    "利用 IoT 传感器进行设备状态实时监测"
  ],
  "cases": [
    {
      "name": "智慧园区运营平台",
      "description": "实现设备监测、能耗分析、工单流转的一体化管理。"
    }
  ],
  "media": []
}
```

## 知识库导入方式

如果需要替换为正式展会资料，可直接编辑 `data/knowledge_base.json`：

1. 修改 `theme` 中的企业名称和标语
2. 更新 `sections` 中的图文视频内容
3. 将集团业务范围、服务特色、先进技术、成功案例、未来规划写入 `entries`
4. 保持 JSON 结构合法，重启 Flask 服务即可生效

也可以通过管理台直接导入：

1. 打开 `/admin`
2. 粘贴完整 JSON
3. 点击“保存知识库”
4. 返回前台刷新页面查看效果

## 本地图片和视频素材接入

当前系统已经预留本地素材接入位，后续你提供正式素材时可直接这样替换：

1. 图片素材：把图片放到 `app/static/` 下，再把 `branding.logo_image` 或 `sections[*].media[*].url` 改成 `/static/文件名`
2. 视频素材：可以使用可嵌入地址，也可以直接使用本地视频文件
3. 若暂时没有正式素材，可将 `url` 留空，前台会显示高质量占位块，不会破版

当前版本已支持本地视频直出：

```json
{
  "type": "video",
  "title": "智慧运营演示视频",
  "url": "/static/demo.mp4",
  "source": "local",
  "mime_type": "video/mp4",
  "poster": "/static/demo-poster.jpg"
}
```

## Logo 替换方式

当前默认使用文本 Logo：

```json
"branding": {
  "logo_type": "text",
  "logo_text": "SIICservice",
  "logo_image": ""
}
```

后续若切换为正式图片 Logo：

```json
"branding": {
  "logo_type": "image",
  "logo_text": "SIICservice",
  "logo_image": "/static/logo.png"
}
```

## 技术实现说明

### 前端

- 使用 `HTML5 + CSS3 + JavaScript`
- 使用 `localStorage` 存储最近 5 条搜索历史
- 使用 Web Speech API 实现语音输入与语音播报
- 采用玻璃拟态与科技蓝主色调设计，适合展会大屏触控场景

### 后端

- 使用 `Flask` 提供页面与 API
- `POST /api/ask` 接收问题并返回知识库匹配结果
- `POST /api/knowledge-base` 支持更新知识库
- 通过关键词、标题命中和中文字符二元组组合完成轻量文本匹配
- 已预留 `LLM_API_URL` 和 `LLM_PROVIDER` 环境变量作为后续大模型接入入口

## 大模型接入预留

当前版本没有直接调用外部大模型，但已经把接入形态预留好：

1. 保持知识库检索作为 grounding 来源
2. 未来可在 `search.py` 中增加基于 `LLM_API_URL` 的二次生成层
3. 页面会自动显示当前处于 `keyword-search` 或 `knowledge-grounded` 模式

这意味着后续可以平滑升级为：

- 先检索知识库候选内容
- 再交给大模型做问答润色和组织表达
- 仍然保持结果受知识库约束

## 展会大屏待机模式

- 页面无操作 45 秒后会自动进入待机轮播模式
- 点击页面右上角按钮也可手动进入
- 点击待机画面任意位置即可退出
- 待机内容来自 `idle_slides`

## 自动讲解模式

- 前台支持手动开启自动讲解
- 自动讲解会按 `guided_tour` 顺序滚动到对应板块
- 若语音功能已开启，会同步播报讲解词
- 观众点击、触摸、移动或键盘操作时会自动退出讲解模式

## 欢迎页与双入口模式

- 页面首次进入会展示欢迎页
- 欢迎页显示当前时间和日期
- 支持两种入口：
  - `领导参观模式`：进入后自动开始讲解
  - `自助参观模式`：进入后保留自由浏览与问答
- 页面顶部会显示当前模式

## 默认演示数据恢复

- 后台 `/admin` 支持一键恢复默认演示数据
- 默认数据来源：`data/knowledge_base.default.json`
- 当前运行数据文件：`data/knowledge_base.json`

## 管理台增强

- `/admin` 现在支持：
  - 表单模式：修改品牌信息、推荐问题、待机轮播、自动讲解脚本
  - JSON 模式：整体替换完整知识库

## 可扩展建议

1. 将示例 `SIIC` 标识替换为正式企业 Logo 图片
2. 接入真实企业宣传视频与案例图片素材
3. 将当前轻量搜索升级为 `spaCy`、`NLTK` 或向量检索方案
4. 为后台增加知识库上传管理页面
5. 若现场有独立大模型服务，可将 `/api/ask` 改为混合检索加 LLM 生成模式

## 注意事项

- 当前示例为可运行演示版本，重点在于结构清晰、便于扩展
- 语音输入兼容性取决于浏览器实现，建议现场使用 Chromium 内核浏览器
- 视频使用的是外部嵌入链接，若展会现场网络受限，建议改为本地视频资源或企业内网可访问资源
