# 上实服务 2026 展会 AI 智能问答模块

这是一个面向 2026 年中国物业协会展会的离线 AI 智能问答展示模块，围绕上实服务“城市运营服务商和美好生活集成商”的品牌定位，提供流式回答、预置 20 道高规格问答、后台题库管理、JSON 导入导出和无网展会现场部署能力。

## 快速启动

```bash
# 一键启动本地展会问答系统
bash install.sh
```

启动后访问终端显示的本地地址，默认端口为 `8000`。

## 文件说明

- `index.html`：展会问答主页面和后台管理入口
- `styles.css`：展会视觉样式，支持桌面和移动端
- `script.js`：离线题库、流式输出、后台增删改、题库导入导出
- `install.sh`：一键启动脚本
- `INSTALL.md`：完整安装手册
- `PITCH_PACKAGE.md`：展会“吹牛皮包”，包含讲解话术、亮点包装和交付清单

## 核心能力

- 无网运行：所有问答和后台数据都在浏览器本地运行
- 流式输出：模拟 AI 智能问答的逐字输出体验
- 后台管理：题目和答案支持新增、编辑、删除
- 题库导入导出：支持 JSON 文件迁移和现场备份
- 展会适配：围绕上实服务城市运营、服务力百强、低空经济、ESG、多业态等主题设计

## 后台入口

页面右上角点击 `后台管理`，即可进入题库后台。

## 离线使用建议

现场电脑提前打开页面并完成题库确认。浏览器会将题库保存到 `localStorage`，断网后仍可展示、检索和流式回答。
