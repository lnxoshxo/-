# 上实服务展会 AI 智能问答模块安装手册

## 1. 交付目标

本模块用于 2026 年中国物业协会展会现场，提供上实服务品牌展示、客户接待、业务介绍和高层洽谈辅助问答能力。系统采用纯静态架构，支持离线运行，适合展台大屏、笔记本、触控屏和内网展示环境。

## 2. 环境要求

- 操作系统：Windows、macOS、Linux 均可
- 浏览器：Chrome、Edge、Safari、Firefox 的现代版本
- 可选运行环境：Python 3
- 网络要求：系统运行无需外网

## 3. 一键安装启动

在项目目录执行：

```bash
# 启动本地静态服务
bash install.sh
```

脚本会自动选择可用端口并打印访问地址，例如：

```text
http://127.0.0.1:8000/index.html
```

打开该地址即可进入展会问答系统。

## 4. 无 Python 环境的直接打开方式

在任意现代浏览器中直接打开 `index.html` 文件，也可以使用核心问答功能。推荐使用 `install.sh` 启动本地服务，以获得更稳定的文件导入导出体验。

## 5. 展会现场部署流程

1. 将整个项目文件夹复制到展会电脑。
2. 执行 `bash install.sh` 启动服务。
3. 打开脚本输出的本地访问地址。
4. 点击 `开始智能问答` 验证流式输出。
5. 点击 `后台管理` 检查 20 道预置题。
6. 使用 `导出题库` 备份现场版本。
7. 断开网络后再次刷新页面验证离线可用。

## 6. 后台管理说明

后台入口位于页面右上角 `后台管理`。

后台支持：

- 新增问题和答案
- 编辑已有问题和答案
- 删除问题
- 查看题目总数、平均答案字数、自定义题数量
- 导入 JSON 题库
- 导出 JSON 题库

## 7. 题库导入格式

支持导入如下 JSON：

```json
{
  "name": "上实服务 2026 展会 AI 智能问答题库",
  "version": "custom",
  "questions": [
    {
      "question": "上实服务如何定义城市运营服务？",
      "answer": "上实服务以专业物业服务为底座，连接城市空间、客户体验、资产价值和社会责任。"
    }
  ]
}
```

也支持直接导入数组：

```json
[
  {
    "question": "问题文本",
    "answer": "答案文本"
  }
]
```

## 8. 数据存储机制

系统将题库保存在浏览器 `localStorage` 中。相同浏览器、相同访问地址下，刷新页面后题库会保留。

重要展会电脑建议每天闭馆前点击 `导出题库` 做备份。

## 9. 恢复默认题库

首页点击 `恢复默认题库`，系统会恢复内置 20 道高规格问答。该操作会覆盖当前浏览器中的自定义题库。

## 10. 常见问题

### 端口被占用

`install.sh` 会自动从 `8000` 开始寻找可用端口。若 `8000` 被占用，会自动尝试后续端口。

### 页面打开为空白

优先使用 Chrome 或 Edge。若仍为空白，执行以下命令检查脚本语法：

```bash
# 检查 JavaScript 语法
node --check script.js
```

### 导入题库失败

请确认 JSON 文件包含 `questions` 数组，且每条记录都有 `question` 和 `answer` 字段。

## 11. 展会验收清单

- 首页可打开
- 20 道默认题显示正常
- 点击问题后答案流式输出
- 搜索关键词可筛选题目
- 快捷提问可快速过滤
- 后台可新增、编辑、删除题目
- 题库可导出
- JSON 题库可导入
- 断网后页面仍可使用
- 移动端或窄屏布局显示正常
