## 10. RAG 实现要求

必须实现标准 RAG：

1. 上传文档
2. 提取文本
3. chunk 切片
4. embedding 向量化
5. 向量存储
6. TopK 检索
7. 基于上下文生成答案
8. 返回答案与引用来源

约束：

1. chunk size 与 overlap 可配置。
2. 支持多文档知识库。
3. 回答时展示引用文档名。
4. 只允许基于知识库回答。
5. 未检索到有效内容时，不允许编造答案。
6. 返回保守回复，例如：
   `当前知识库未检索到足够信息，请联系现场顾问进一步了解。`
