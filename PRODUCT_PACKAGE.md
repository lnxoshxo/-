# 上实服务 2026 展会 AI 智能问答产品安装包说明

## 产品包名称

`siic-service-expo-ai-qa-product-package.zip`

## 产品定位

本产品包是面向 2026 年中国物业协会展会的一键安装版离线 AI 智能问答系统。它将上实服务的品牌定位、城市运营能力、行业荣誉、科技实践和高品质服务表达，封装为可本地启动、可无网运行、可后台维护、可现场演示的完整交付包。

## 包内文件

- `README.md`：快速启动说明
- `INSTALL.md`：完整安装手册
- `PITCH_PACKAGE.md`：展会讲解话术和产品包装说明
- `PRODUCT_PACKAGE.md`：产品安装包说明
- `index.html`：展会问答系统主页面
- `styles.css`：展会系统样式
- `script.js`：离线问答、流式输出和后台题库逻辑
- `install.sh`：Linux/macOS/Git Bash 一键启动脚本

## 一键安装方式

解压产品包后，在目录中执行：

```bash
# 启动产品
bash install.sh
```

启动成功后打开终端输出的地址：

```text
http://127.0.0.1:8000/index.html
```

## 无网使用方式

系统为纯静态离线产品，启动后无需外网即可使用。题库保存在浏览器本地 `localStorage`，现场可新增、编辑、删除、导入和导出题库。

## 展会交付验收

- 能打开首页
- 能看到 20 道默认高规格问答
- 点击问题后答案按流式输出
- 能通过关键词搜索问题
- 能使用快捷提问筛选主题
- 能进入后台管理
- 能新增、编辑、删除题目
- 能导出 JSON 题库
- 能导入 JSON 题库
- 断网后仍可运行

## 推荐交付话术

这是一个可离线运行的上实服务展会 AI 智能问答产品包，现场一键启动，无需部署服务器，无需连接外网，既能展示 AI 交互体验，又能确保品牌口径统一、内容安全可控、题库现场可维护。
