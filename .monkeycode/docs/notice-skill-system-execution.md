# 关于印发《物业集团信息化技能体系执行方案》的通知（发文版）

各区域公司、各项目公司、集团各职能中心：

为全面提升集团信息化治理能力，推进数字化建设标准化、体系化、可审计化运行，现将《物业集团信息化总经理全闭环 Skill 技能体系》作为集团信息化统一执行标准予以印发，请遵照执行。

## 一、执行目的
1. 建立覆盖战略规划、项目建设、运维安全、数据运营、预算采购、外部协同的全业务闭环管理体系。
2. 建立总部、区域、项目三级统一的信息化管理语言与执行标准。
3. 实现“目标可分解、过程可追踪、结果可量化、责任可追溯”。

## 二、执行范围
1. 集团总部信息化部（数字信息中心 / 科技信息化中心）。
2. 各区域公司信息化团队。
3. 各项目公司 IT 支撑岗位及相关业务协同岗位。

## 三、执行依据文件
1. `.monkeycode/docs/it-gm-skill-system.md`

## 四、组织分工
1. 信息化部总经理：对体系落地负总责，主抓战略、预算、重大项目与重大应急。
2. 各条线负责人：对条线执行成效负责，确保流程、模板、数据、复盘闭环。
3. PMO：负责跨部门协同、项目组合治理、里程碑跟踪与例会机制。
4. 区域公司：负责本区域落地推广、差异化适配与执行反馈。
5. 项目公司：负责现场执行、工单闭环、培训推广与一线问题上报。

## 五、执行要求
1. 自本通知发布之日起，信息化相关工作必须按 Skill 编号进行任务拆解、执行与归档。
2. 所有项目必须执行“立项评审 -> 里程碑验收 -> 上线切换 -> 后评估复盘”闭环。
3. 所有重大故障与安全事件必须执行统一应急指挥机制和标准复盘模板。
4. 所有预算申请必须可追溯至项目与 Skill，未完成 ROI 复盘的同类项目不得扩投。
5. 各单位须建立周检、月检、季检机制，并按规定报送执行结果。

## 六、阶段计划
1. 第一阶段（发布后 7 天内）：完成宣贯培训、角色映射、RACI 发布。
2. 第二阶段（发布后 30 天内）：完成首轮试运行与流程抽查，形成整改清单。
3. 第三阶段（发布后 90 天内）：实现全量项目纳管，形成季度评估报告。

## 七、考核机制
1. 将执行情况纳入信息化条线季度 KPI 与年度绩效。
2. 对执行优秀单位进行通报表扬；对长期未达标单位实施专项督导。
3. 对重大流程缺失、重大风险漏报、重大预算失控等情况，按集团制度追责。

## 八、报送要求
1. 各区域公司每季度最后一个工作日前，报送《季度执行通报》及附件台账。
2. 报送路径：`集团信息化管理平台 -> 技能体系执行专栏`。
3. 报送材料须确保数据真实、证据齐套、结论可复核。
