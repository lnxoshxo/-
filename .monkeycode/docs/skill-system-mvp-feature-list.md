# SkillOS MVP 功能与页面清单（可开工）

## 1. MVP 范围（首期 6-8 周）
- 目标：跑通三条主链路（平台上线、故障应急、预算编制）。
- 原则：先闭环、再扩展；先流程可用、再智能化。

## 2. MVP 功能清单

| 模块 | 功能 | 优先级 | 验收标准 |
|---|---|---|---|
| Skill 字典 | Skill CRUD、版本管理、上下游依赖 | P0 | 108 个 Skill 可入库与检索 |
| 流程中心 | 流程定义、节点流转、SLA 超时升级 | P0 | 三条主流程可跑通 |
| 任务中心 | 自动派单、状态流转、催办 | P0 | 自动派单成功率>=98% |
| 模板中心 | 模板实例化、文档上传、版本留痕 | P0 | 12 个模板可用 |
| 权限中心 | 组织/角色/RACI 授权 | P0 | 关键 Skill 唯一 A 生效 |
| 驾驶舱 | KPI 看板（交付、稳定、安全、预算） | P1 | 4 类 KPI 自动汇总 |
| 审计中心 | 操作日志、证据归档检索 | P1 | 审计抽查可追溯 |

## 3. 页面清单

| 页面 | 主要用户 | 核心能力 |
|---|---|---|
| 登录与组织切换 | 全员 | SSO 登录、组织上下文切换 |
| Skill 字典列表 | PMO/DIR | 按域、模块、编号筛选 |
| Skill 编辑页 | PMO | 编辑输入/输出/KPI/RACI |
| 流程设计页 | PMO/架构 | 配置节点、网关、SLA |
| 流程运行看板 | GM/DIR | 实例状态、阻塞、超时 |
| 我的待办 | MGR/SPC | 接单、提交、回退、评论 |
| 交付物中心 | MGR/SPC | 模板实例、上传、签核 |
| KPI 驾驶舱 | GM/DIR | 趋势图、组织钻取 |
| 风险台账页 | DIR/MGR | 风险登记、分级、闭环 |
| 审计检索页 | 审计/合规 | 按实例导出证据链 |

## 4. MVP 用户故事（关键）
1. 作为 PMO，我可以从 Skill 字典发起一个流程实例，并自动分派任务。
2. 作为运维经理，我可以在 P1 告警触发时自动拉起应急流程。
3. 作为财务BP，我可以在预算流程中关联项目与历史 ROI。
4. 作为总经理，我可以查看跨组织 KPI 看板和超时风险项。

## 5. 里程碑建议
1. 第 2 周：Skill 字典 + 权限中心可用。
2. 第 4 周：三条流程可跑通。
3. 第 6 周：模板与审计留痕可用。
4. 第 8 周：KPI 驾驶舱上线并验收。
