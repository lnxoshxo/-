# 物业集团信息化总经理技能体系模板库（可直接复用）

## 使用说明
- 本模板库与 `it-gm-skill-system.md` 的 Skill 编号一一映射。
- 所有模板均满足：可成文、可归档、可审批、可审计。
- 建议命名规范：`YYYYMMDD-模板名-项目或事项名称-vX.Y`。

## 模板落地建议
1. 每个模板绑定一个主责岗和一个复核岗。
2. 每次输出必须有版本号和审批签名。
3. 模板填写后进入统一文档库，纳入审计抽查。
4. 每季度根据复盘结果更新模板字段。
