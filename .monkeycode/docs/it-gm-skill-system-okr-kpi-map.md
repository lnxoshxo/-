# 物业集团信息化总经理 Skill 体系 OKR/KPI 映射包

## 1. 使用范围
- 用于信息化部年度目标制定、季度绩效考核、干部述职与人才盘点。
- 与主文档 `it-gm-skill-system.md` 及模板库联动使用。

## 2. 映射原则
1. 一个 O（目标）至少映射 2 个 KR（关键结果）。
2. 每个 KR 至少映射 2-4 个 Skill 编号，保证可执行。
3. KPI 必须满足可量化、可归因、可复盘。
4. 考核采用“结果+过程”双轨：结果 70%，过程 30%。

## 7. 述职汇报结构（总经理/中层通用）
1. 本期目标与达成情况（对齐 O/KR）。
2. 关键成果（按战略、交付、稳定、安全、经营、成本六维）。
3. 典型闭环案例（至少 1 个）。
4. 未达标项与根因。
5. 下周期纠偏计划与资源诉求。

## 8. 推行步骤（建议）
1. 第 1 周：确认年度 OKR 与季度 KPI 口径。
2. 第 2 周：完成各岗位权重校准并发布评分卡。
3. 第 3 周：上线周报/月报/季报节奏。
4. 第 4 周：组织首次绩效复盘，修订不合理指标。
