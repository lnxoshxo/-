# 物业集团信息化总经理技能体系（部门执行版检查清单）

## 使用说明
- 本清单用于将 `it-gm-skill-system.md` 的 Skill 体系落到日常管理动作。
- 使用频率：周检（执行层）、月检（管理层）、季检（总经理）。
- 检查原则：有清单、有证据、有责任人、有时限、有复盘。

## D. 三大典型闭环检查模板

### D1 新平台建设上线闭环
1. 立项依据是否充分：`3.01.03`。
2. 计划与资源是否锁定：`3.02.01`。
3. 质量门禁是否达标：`4.02.03`。
4. 上线切换是否有回退：`3.02.04`。
5. 上线后是否有运营改进：`4.03.01`。
6. 是否完成后评估：`3.03.02`。

### D2 故障应急闭环
1. 告警是否及时：`5.02.01`。
2. 指挥是否统一：`9.03.01`。
3. 恢复时长是否达标：`5.02.03`。
4. 对外沟通是否一致：`9.03.02`。
5. 是否修订制度并复演：`9.03.04`、`6.03.02`。

### D3 预算编制闭环
1. 是否基于战略路线图：`1.02.04`。
2. 需求是否经价值评审：`3.01.02`。
3. 预算是否可追溯到项目：`8.01.01`。
4. 执行中是否滚动调整：`8.01.02`。
5. 年终是否完成 ROI 复盘：`8.01.04`。

## E. 红线机制（必须升级总经理）
- 重大安全事件或连续中断超过 SLA 红线。
- 预算超支超过 10% 且无审批。
- 重大项目关键里程碑连续两次延期。
- 供应商违约影响核心系统稳定。
- 对外监管通报类问题。
