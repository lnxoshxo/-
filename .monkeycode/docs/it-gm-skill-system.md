# 物业集团信息化部总经理全闭环 Skill 技能体系（完整版）

## 0. 体系说明
- 适用组织：集团总部 + 区域公司 + 项目公司三级架构。
- 适用业态：住宅、商业、写字楼、产业园、学校、医院、公建物业。
- 架构规则：9 个一级技能域，27 个二级模块，108 个三级 Skill。
- 编号规则：`一级域.二级模块.三级技能`，例如 `1.01.01`。
- 闭环规则：所有 Skill 满足 `输入 -> 处理 -> 输出 -> 流转`，可串联形成端到端闭环。

## 3. 技能闭环逻辑（通用编排）

5. 上线运营：调用 `3.02.04 -> 4.03.01 -> 7.03.01`。
6. 复盘优化：调用 `3.03.02 -> 1.03.02 -> 8.01.04`。
