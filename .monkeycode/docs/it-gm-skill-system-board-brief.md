# 物业集团信息化总经理技能体系（董事会汇报版）

## 1. 汇报目标
- 用一套可量化、可授权、可审计的 Skill 体系，保证集团信息化建设从战略到执行全流程闭环。
- 支撑集团多业态、三级组织架构下的信息化统一治理与差异化落地。

## 2. 体系全景（9+27+108）
- 一级域：9 大治理域（战略、组织、项目、产品、基础设施、安全、数据、预算采购、协同创新应急）。
- 二级模块：27 个管理模块。
- 三级 Skill：108 个标准化执行单元。
- 闭环逻辑：输入 -> 评估 -> 执行 -> 验收 -> 复盘 -> 优化 -> 再规划。

## 3. 管理价值
| 价值项 | 体现方式 | 关键结果 |
|---|---|---|
| 战略落地 | 战略目标映射到 Skill 编号与项目池 | 战略项按期达成率提升 |
| 组织提效 | 授权矩阵 + 跨部门接口人机制 | 决策时长下降、协同效率提升 |
| 投资可控 | 预算、采购、ROI 闭环治理 | IT 投入产出可核验 |
| 风险可控 | 安全、故障、危机应急标准化 | 重大事件响应更快、影响更小 |
| 数据增值 | 数据标准、指标体系、BI 运营 | 管理决策从经验驱动转向数据驱动 |

## 4. 年度重点抓手
1. 战略：完成 `1.02.01`、`1.02.04`，形成 3-5 年战略与年度路线图。
2. 交付：以 `3.01.*` + `3.02.*` 建立项目统一里程碑治理。
3. 稳定：以 `5.02.*` + `9.03.*` 建立故障应急指挥体系。
4. 安全：以 `6.01.*` + `6.02.*` + `6.03.*` 建立合规和事件闭环。
5. 经营：以 `7.02.*` + `7.03.*` 建立经营驾驶舱和专题分析机制。
6. 价值：以 `8.01.04` 建立年度 ROI 复盘机制。

## 5. 总经理必亲自操盘事项
- 战略与报审：`1.02.01`、`1.03.04`
- 重大立项与组合：`3.01.03`、`3.03.03`
- 年度预算：`8.01.01`
- 重大应急：`9.03.01`、`6.03.01`
- 战略创新：`9.02.03`

## 6. 董事会关注 KPI（建议）
| 维度 | KPI |
|---|---|
| 交付 | 里程碑按期率、一次验收通过率 |
| 稳定 | SLA 达标率、MTTD、MTTR |
| 安全 | 高危漏洞修复率、重大安全事件数 |
| 经营 | 核心指标准确率、看板使用率 |
| 投入产出 | 预算偏差率、采购节约率、ROI |

## 7. 里程碑节奏（示例）
- Q1：战略路线图、预算定版、重点项目立项。
- Q2：核心平台建设与首批上线，安全与监控基线达标。
- Q3：多业态复制推广，数据驾驶舱覆盖主要经营场景。
- Q4：ROI 复盘、组织能力评估、下一年度滚动规划。
