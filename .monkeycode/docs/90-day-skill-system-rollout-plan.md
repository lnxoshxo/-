# 信息化技能体系 90 天落地推进台账（甘特化）

## 1. 目标与原则
- 目标：90 天内完成体系发布、试运行、全量纳管与季度复盘，形成可持续运行机制。
- 原则：周周有进展、月月有验收、关键节点可审计。

## 2. 角色与职责
- `GM`：信息化部总经理，重大决策与里程碑验收。
- `PMO`：推进总控、周报汇总、风险升级。
- `DIR`：条线负责人，模块落地负责人。
- `MGR/SPC`：执行与材料输出。

## 8. 最终验收标准（Day 90）
1. 体系覆盖：重点项目纳管率 >= 95%。
2. 执行合规：核心模板使用率 >= 95%。
3. 稳定能力：SLA 达标率 >= 99.9%。
4. 安全能力：高危漏洞按时修复率 >= 98%。
5. 经营管理：季度 KPI 完整上报率 = 100%。
6. 复盘能力：90 天整改项按期关闭率 >= 90%。
